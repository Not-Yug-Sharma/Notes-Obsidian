# C Tokens
C tokens are the smallest units of a C program that hold meaning to the compiler. The different types of C tokens are:
	1. Identifier:
		- Identifiers are the names of variables, constants, functions, arrays etc.
		- They are user defined.
		- There are rules to defining a new Identifier. They are:
			- An identifier should only consist of alphabets(a-z or A-Z), numbers(0-9) and underscore( _ ).
			- An identifier must start with an alphabet(a-z or A-Z).
